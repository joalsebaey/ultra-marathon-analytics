# Raw Data

The raw dataset is not included in this repository due to its large file size (~500MB).

## Download Instructions

1. Go to: [https://www.kaggle.com/datasets/aiaiaidavid/the-big-dataset-of-ultra-marathon-running](https://www.kaggle.com/datasets/aiaiaidavid/the-big-dataset-of-ultra-marathon-running)
2. Download: `TWO_CENTURIES_OF_UM_RACES.csv`
3. Place it in this folder: `data/raw/`
4. Run the notebook: `notebooks/ultra_marathon_data_transformation.ipynb`
